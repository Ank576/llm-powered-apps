# Credit Score Simulator

AI-Powered Credit Score Impact Analyzer

An intelligent Streamlit application that simulates and predicts credit score changes based on financial behaviors and decisions.

## Features

- Score Impact Simulation: Visualize how payment history, credit utilization, and credit inquiries affect your score
- Behavioral Analysis: AI-powered insights using Perplexity LLM on financial decisions
- Scenario Planning: Compare multiple financial scenarios and their credit impacts
- RBI Compliance: Aligned with RBI guidelines on credit assessment
- Interactive Dashboard: Real-time visualization of credit score changes

## Tech Stack

- Frontend: Streamlit
- LLM: Perplexity API
- Backend: Python 3.11+
- Compliance: RBI Guidelines

## Installation

cd credit-score-simulator
pip install -r requirements.txt

## Running the App

streamlit run app.py

## Important Disclaimer

This tool is for educational and demo purposes only. It does not replace professional credit counseling or official credit assessments from authorized financial institutions.
